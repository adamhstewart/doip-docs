# Saves and Checks
## Ability Checks
```sh
# Format: !check <skill/ability>
!check arcana
!check wisdom
!c perc
```

## Saving Throws
```sh
# Format: !save <ability>
!save wisdom
!s dex
```

## Death Saves
```sh
!save death        # Rolls a death save.
```

To reset Avrae's death save counters manually (automatically done when you heal HP):
```sh
!game deathsave reset
```
